
# Calendar

## Run Locally

Clone the project

```bash
  git clone git@github.com:Sajal0208/zenskar-calendar.git
```

Go to the project directory

```bash
  cd zenskar-calendar
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
